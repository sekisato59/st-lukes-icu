ACNS 標準クリティカルケアEEG用語集 2021年版

American Clinical Neurophysiology Society's Standardized Critical Care EEG Terminology: 2021 Version

ACNS 2021 J Clin Neurophysiol 2021年1月 神経系
セクション A　EEG背景波
EEG Background

A1 対称性（Symmetry）

対称性の分類

背景波の対称性は3段階で評価する。半球間の非対称性は50%以上の電位差、または1Hz以上の周波数差で「著明な非対称性（Marked asymmetry）」と定義される。以下のいずれかの特徴が非対称の場合、左右半球を独立して記載する（A2〜A10）。

分類定義
対称（Symmetric）両側で電位・周波数がほぼ等しい
軽度の非対称（Mild asymmetry）適切な参照電極記録で一貫した＜50%の電位差、または0.5〜1 Hzの周波数差
著明な非対称（Marked asymmetry）適切な参照電極記録で≥50%の電位差、または＞1 Hzの周波数差

NOTE: 電位の計測は波のピーク間（peak to trough）で行い、標準10-20縦ビポーラ導出を基本とする。

A2–A3 優位背景周波数と後頭部優位律動

優位背景周波数（Predominant Background Frequency）

覚醒時または刺激後の状態で評価する。2〜3つの周波数帯が等しく優位な場合はそれぞれ記載する。

Beta（ベータ）
>13 Hz
Alpha（アルファ）
8〜13 Hz
Theta（シータ）
4〜8 Hz
Delta（デルタ）
<4 Hz

後頭部優位律動（Posterior Dominant "Alpha" Rhythm）

後頭部優位律動は、開眼で減衰することを確認する（"alpha squeak"を避けるため、眼球閉鎖後1秒以上待ってから周波数を測定する）。存在する場合は0.5 Hz単位で周波数を記載する。

存在（Present：周波数を記載）/ 欠如（Absent）/ 不明（Unclear）の3択で分類する。

A4 連続性（Continuity）

連続性の分類

連続性は、記録中に「減衰（Attenuation）」または「抑制（Suppression）」が占める割合で分類する。減衰とは≥10 μVかつ背景電位の＜50%の低電位期間、抑制とは＜10 μVの低電位期間と定義される。バーストは平均≥0.5秒かつ少なくとも4相（baseline crossings）を持つ必要がある。

分類定義
Continuous（連続）減衰/抑制が記録の＜1%
Nearly Continuous（ほぼ連続）減衰/抑制が記録の1〜9%
Discontinuous（不連続）減衰/抑制が記録の10〜49%
Burst attenuation / Burst suppression減衰/抑制が記録の50〜99%
Attenuation / Suppression減衰/抑制が記録の＞99%

「Suppression-burst」は「Burst-suppression」の同義語である。バーストの局在・典型的持続時間・バースト間隔・Highly Epileptiform Bursts の有無も記載する。

A5 反応性（Reactivity）

反応性の定義と評価

刺激に対する脳波活動の変化（電位・周波数の変化、活動の減衰を含む）を反応性と定義する。筋電図アーチファクトや眼球運動は反応性とはみなさない。段階的に強度を上げた複数の刺激を用いた標準的な検査プロトコルを強く推奨する。

分類解説
Reactive（反応あり）刺激に対して脳波変化がある
Unreactive（反応なし）刺激に対して脳波変化がない。2回目の標準的な有害刺激（胸骨圧迫・鼻腔スラブなど）で確認し、鎮静・筋弛緩の影響も注記する
SIRPIDs only刺激誘発性のリズミック・周期・てんかん様放電のみが反応性として見られる場合（SI-RDA・SI-PDs・SI-SW・SI-seizures・SI-bursts・SI-IIC・SI-BIRDsを含む）
State changesが存在すること自体は反応性の存在を示すが、反応性があってもState changesが存在しない場合がある（反応が＜60秒の場合）。

A6–A7 State Changes と CAPE

State Changes（状態変化）

覚醒度・刺激に関連した少なくとも2種類の持続的（各≥60秒）な背景波パターンが存在する場合に「Present」とする。刺激によってより覚醒した状態へ移行できることが条件。より覚醒/刺激された状態が「報告される背景波」として扱われる。K複合体とスピンドルが存在する場合は正常/異常を区別し記述する。

CAPE（脳症の周期性交代パターン）

Cyclic Alternating Pattern of Encephalopathy（CAPE）とは、RPPsを含む可能性のある2つの背景波パターンが、各パターン≥10秒間持続し、少なくとも6サイクル以上規則的に自然に交代するパターンを指す。1サイクルは両方のパターンを含む期間として定義される。

各パターンが＞60秒続く場合は必然的にState changesとなる。CAPEがあってもState changesがない場合がある（一方のパターンが＜60秒の場合）。

AP勾配（Anterior-Posterior Gradient）：前方に高周波数・低電位、後方に低周波数・高電位の勾配が少なくとも1分間持続する場合に「Present」。逆方向の場合は「Present, but reversed」。

SUMMARY セクション A のキーメッセージ

① 対称性：適切な参照電極記録で≥50%電位差またはを1Hz以上の周波数差が「著明な非対称」。左右別々に記載する。
② 連続性：Burst-suppressionは減衰/抑制が50〜99%。Suppressedは＞99%。バースト間抑制は平均≥0.5秒・4相以上が必要。
③ 反応性：段階的な複数刺激のプロトコルを強く推奨。SIRPIDsのみが反応として出現する場合は別カテゴリーとして記録する。
④ CAPE：2パターンが各≥10秒間持続し自然交代を少なくとも6サイクル繰り返す脳症特有のパターン。CAPEがあってもState changesがない場合がある。
⑤ 電位：Suppressed（＜10 μV）とLow（＜20 μV）は区別する。電位測定はピーク間で行う。頭蓋欠損があればブリーチ効果を必ず記載。
セクション B　散発性てんかん型放電
Sporadic Epileptiform Discharges

B 定義と出現頻度の分類

てんかん型放電の定義

散発性てんかん型放電とは、非リズミック・非周期性のスパイク・ポリスパイク・鋭波を指す。いずれもリズミックあるいは周期的でなく、孤立した形で出現するものである。

波形定義
スパイク（Spike） 背景波から明確に区別される過渡波で、尖頭を持ち、通常の時間軸・感度で20〜70 ms未満の持続時間。EEGベースラインで測定
鋭波（Sharp wave） スパイクと同様の定義だが持続時間が70〜200 ms。通常、下降脚が急峻（上昇脚より急峻）で、後続緩徐波を伴うことが多い
ポリスパイク（Polyspike） 2つ以上のスパイクが放電間隔なしに連続し持続時間＜0.5秒のもの。≥0.5秒ならBIRDsまたはバースト内の高てんかん型バーストに該当

出現頻度の分類

てんかん型放電の出現頻度（スパイク・ポリスパイク・鋭波の合計）は以下の4段階で分類する。Abundant（豊富）の場合は、10秒エポックあたりの推定平均・最大スパイク数の記載が有用。

Abundant（豊富）
≥1個/10秒だが周期性なし
Frequent（頻繁）
≥1個/分〜＜1個/10秒
Occasional（時折）
≥1個/時〜＜1個/分
Rare（稀）
＜1個/時
セクション C　リズミック・周期パターン（RPPs）
Rhythmic and Periodic Patterns

C — 主用語1 局在の分類（G / L / BI / UI / Mf）

局在の分類（Main Term 1）

RPPsはすべて2つの主用語と修飾語から構成される。主用語1はパターンの局在を、主用語2はパターンの種類を示す。

略語名称定義
G Generalized（汎化性） 両側半球同期・対称性のパターン。シフティング優位性（非対称性に基づく）またはシフティング非同期性（タイミングに基づく、一貫性なし）のパターンも含む
L Lateralized（側在性） 片側のみ（Unilateral）、または一側優位で両側に明確かつ一貫して出現（Bilateral asymmetric）、または一側が一貫して先行する（Bilateral asynchronous）。焦点性・領域性・半球性を含む
BI Bilateral Independent（両側独立性） 各半球に独立した側在性パターンが同時に存在（重複）。2つのパターンが独立かつ非同期
UI Unilateral Independent（一側独立性） 同一半球内の独立した2つの周期性またはリズミックパターンが同時に存在
Mf Multifocal（多焦点性） 少なくとも3つの独立した側在性パターンが各半球に1つ以上、同時に存在

C — 主用語2 パターンの種類（PDs / RDA / SW）

PDs（周期性放電）、RDA（リズミックデルタ活動）、SW（スパイク・波複合）

略語名称定義のポイント
PDs Periodic Discharges
（周期性放電）		 ほぼ均一な形態・持続時間の波形が明確な放電間隔（interdischarge interval）をもって繰り返す。「ほぼ規則的な間隔」＝cycle長の変動が過半数（＞50%）のcycleペアで＜50%。少なくとも6サイクル継続
RDA Rhythmic Delta Activity
（リズミックデルタ活動）		 ほぼ均一な形態・持続時間の波形が放電間隔なしに繰り返す（0.5〜4.0 Hz）。cycle長の変動が過半数のcycleペアで＜50%。少なくとも6サイクル継続
SW Spike-and-wave /
Sharp-and-wave		 スパイク（ポリスパイクまたは鋭波）と緩徐波が一貫した関係で規則的に交互に繰り返す（スパイク-波-スパイク-波-スパイク-波のパターン）。少なくとも6サイクル継続。スパイク波複合と次のスパイク波複合の間に間隔がない
PDs・SWで典型周波数が＞2.5 Hzの場合、持続が＜10秒なら適用可能（定義上「very brief」）。持続≥10秒なら電気的発作（Criterion A）として扱うべき。いかなるRPPも典型周波数が＞4 Hzにはなれない（＞4 Hz かつ ≥0.5秒ならBIRDs、または≥10秒なら電気的発作となる）。

C — 主修飾語 主修飾語（Main Modifiers）

主修飾語一覧

主修飾語はほぼすべてのEEG現象に適用できる。複数の修飾語が存在する場合はそれぞれ記載する。

修飾語内容
Prevalence（出現率） Continuous（≥90%）/ Abundant（50〜89%）/ Frequent（10〜49%）/ Occasional（1〜9%）/ Rare（＜1%）
Duration（持続時間） Very long（≥1時間）/ Long（10〜59分）/ Intermediate（1〜9.9分）/ Brief（10〜59秒）/ Very brief（＜10秒）
Frequency（周波数） 0.5, 1, 1.5, 2, 2.5 Hz（またはvery briefなら3, 3.5, 4 Hz）で最小・最大・典型値を記載
Number of phases（相数） 1 + ベースラインクロス数（1, 2, 3, または＞3）
Sharpness（鋭さ） Spiky（＜70 ms）/ Sharp（70〜200 ms）/ Sharply contoured（sharp slope but ≥200 ms）/ Blunt（平滑・正弦波状）。PDs・SWの spike/sharp component に適用。RDAには不適用
Voltage（電位） Very low（＜20μV）/ Low（20〜49μV）/ Medium（50〜149μV）/ High（≥150μV）。PDsの最高電位成分で計測
SI / ST（刺激誘発/刺激終止） Stimulus-Induced（SI-）：刺激により誘発または増悪。Stimulus-Terminated（ST-）：刺激により終止または減弱。「SIRPIDs」は総称（SI-RDA, SI-PDs, SI-SW, SI-IIC, SI-BIRDs等を含む）

Evolution（経過）/ Fluctuating（変動）/ Static（静的）

経過の修飾語は周波数・形態・局在の変化を記述する。Evolving（経過中）とは、少なくとも2つの一方向的な連続変化（各周波数は3サイクル以上持続、変化量は≥0.5 Hz）が5分以上変わらずに続いた後で生じるものを指す。

分類定義
Evolving（経過中） 周波数・形態・局在のいずれかに少なくとも2つの一方向的変化。変化間隔は1分超（Fluctuatingとの区別）。なお電位のみの変化はEvolvingに該当しない
Fluctuating（変動） 周波数・形態・局在のいずれかに≥3回の変化が、各変化が1分以内に起きるが、Evolvingには該当しないもの（例：1 Hzと1.5 Hzの間を繰り返す）
Static（静的） EvolvingにもFluctuatingにも該当しない

Plusモディファイア（＋）と Extreme Delta Brush（EDB）

Plus（＋）修飾語は、パターンをより発作に似た様子（ictal-appearing）にする付加的特徴を示す。PDsおよびRDAにのみ適用（SWには適用しない）。

修飾語定義
＋F（Fast）重畳する（superimposed）速波活動（シータ以上）。RDAとPDs両方に適用可能
＋R（Rhythmic）重畳する準リズミックデルタ活動。PDsにのみ適用
＋S（Sharp/Spiky）重畳する鋭波・スパイク、または鋭く輪郭されている。RDAにのみ適用。鋭い輪郭・鋭波・スパイクが10秒に1回以上（SWパターンの一部としてではなく）出現が必要
Extreme Delta Brush（EDB）：＋Fのサブタイプ。RDA＋F（fast activityがデルタ波に定型的な関係で出現）またはPDs＋F（PDsが単一の鈍いデルタ波で、stereotypedのfast activityが各波に重畳）。Continuous/Abundant（≥50%）なら「Definite EDB」、Frequent/Occasional（1〜49%）なら「Possible EDB」。

SUMMARY セクション C のキーメッセージ

① 命名規則：RPPsは「主用語1（局在）＋主用語2（種類）＋修飾語」で構成される（例：LPDs＋F, GPDs, GRDA＋S）。
② PDs vs RDA：PDs（周期性放電）は放電間隔あり、RDA（リズミックデルタ）は放電間隔なし。両者が同時に存在する場合はPDs＋Rとして記載。
③ 周波数制限：いかなるRPPも典型周波数が＞4 Hzにはなれない。＞4 Hz かつ ≥0.5秒ならBIRDsに、≥10秒なら電気的発作に分類される。
④ Plusモディファイア（＋）：パターンをより発作様にする付加特徴。＋F（速波重畳）・＋R（リズミックデルタ重畳）・＋S（鋭波/スパイク重畳）のサブタイプに分類。
⑤ EDB：抗NMDA受容体脳炎に特徴的とされる。RDA＋FまたはPDs＋Fのサブタイプで、fast activityがデルタ波に定型的関係で出現するものが「Definite EDB」。
セクション D　電気的・電気臨床的発作 NEW 2021
Electrographic and Electroclinical Seizures

D1 電気的発作（ESz：Electrographic Seizure）

電気的発作（ESz）の定義（ザルツブルク基準に基づく）

電気的発作（ESz）は以下のいずれかを満たすものとして定義される。明確な臨床的相関（ECSz）を必ずしも必要としない。

A

Criterion A：てんかん型放電（EDs）が10秒間で平均＞2.5 Hz（＞25個/10秒）。技術的には「てんかん型」でない鋭く輪郭された放電（例：10秒間で＞25個の鋭く輪郭された放電）でも発作になり得る。

または
B

Criterion B：明確な経過（evolution）を伴い≥10秒続くいかなるパターンも電気的発作に該当する。

「10秒ルール」の採用・削除については著者らの間で大きな議論があったが、明確なコンセンサスが得られなかったため現状維持とした。

D2 電気的てんかん重積（ESE：Electrographic Status Epilepticus）

電気的てんかん重積（ESE）の定義

ESEは以下のいずれかを満たすものとして定義される。

≥10分間
連続するESz
（continuous）
≥20%
任意の60分間
に占めるESzの割合

10分のカットオフは、意識障害を伴う焦点性てんかん重積状態のILAE定義（Trinka 2015）に一致する。20%のカットオフは前バージョンの50%から引き下げられ、専門家コンセンサスと重症小児の研究に基づいている（最大時間当たり発作負荷が＞20%で神経学的悪化リスクが有意に高かった）。

D3–D4 電気臨床的発作（ECSz）と電気臨床的てんかん重積（ECSE）

電気臨床的発作（ECSz）

ECSzはいかなるEEGパターンでも以下のいずれかを満たせば定義される。ESzとしての持続基準（≥10秒）を必要としない。

基準内容
基準 a 明確な臨床的相関がEEGパターンに時間的に連動している（任意の持続時間）。眼瞼ミオクローヌス・眼球偏位・四肢ジャーキングなど。臨床症状がパターン存在時に認められ、非存在時に消失することが必要。
基準 b EEGおよび臨床状態の両者が静脈内抗てんかん薬により改善する（任意の持続時間）。
静的な1 Hz PDsに顔面チックの臨床的相関がある場合：ESzには該当しないが（evolution なし）、ECSzには該当する。多くの発作は「electrographic」かつ「electroclinical」の両方に該当し、両方の用語で報告すべきである。

電気臨床的てんかん重積（ECSE）と Possible ECSE

ECSEはECSzが≥10分間連続または任意の60分間の≥20%を占めるもの。既知のてんかん性脳症を持つ患者の場合は、EEGパターンがベースラインからの変化（てんかん型放電の増加など）と臨床状態の悪化を示す必要がある。

Possible ECSE（4b）：IICに該当するRPPが≥10分間連続または60分の≥20%に存在し、抗てんかん薬の静注でEEGは改善するが臨床的改善はないもの。ザルツブルク基準における「possible NCSE」とほぼ同義。ESz/ESEには該当するパターンは含めない。

SUMMARY セクション D のキーメッセージ

① ESzの2基準：Criterion A（EDs ＞2.5 Hz / 10秒）またはCriterion B（明確なevolutionが≥10秒）のいずれかを満たせばESz。
② ESEのカットオフ：連続≥10分 または 任意60分間の≥20%。前バージョンの50%閾値から20%へ引き下げ（臨床的意義に基づく）。
③ ECSzはESzを前提としない：持続が10秒未満でも、明確な臨床的相関または抗てんかん薬での改善があればECSzに該当する。
④ Possible ECSE：IICパターンで抗てんかん薬によるEEG改善はあるが臨床改善がないもの。「Possible NCSE」（ザルツブルク基準）とほぼ同義。
⑤ SI-prefix：ESz・ECSz・ESE・ECSEが刺激誘発性（reproducibly brought about or exacerbated by alerting stimulus）であれば「SI-」の接頭辞を付ける。
セクション E　BIRDs NEW 2021
Brief Potentially Ictal Rhythmic Discharges

E BIRDsの定義と分類

BIRDsの定義（Yoo JY et al. 2017 に基づく）

BIRDs（Brief Potentially Ictal Rhythmic Discharges）は、焦点性または汎化性（G, L, BI, UI, Mf）のリズミック活動で＞4 Hz、少なくとも6サイクル、≥0.5秒〜＜10秒の持続を示し、以下の除外基準を満たすパターンである。

除外基準（以下のいずれにも該当しない）：正常パターンまたは良性変異（benign variant）でない・バースト抑制/バースト減衰の一部でない・明確な臨床的相関がない

BIRDsであるためには以下の少なくとも1つの特徴が必要：

分類特徴
Definite BIRDs（a） Evolution（経過）を伴う「evolving BIRDs」
Definite BIRDs（b） 同一患者の発作間欠期てんかん型放電または発作と類似した形態・局在
Possible BIRDs（c） 鋭く輪郭されているが（a）も（b）もない
技術的には全BIRDsが「very brief」（＜10秒）。Paroxysmal fast activity（≥0.5〜＜10秒）もBIRDsの一形態（Generalized Paroxysmal Fast Activity: GPFAなど）。BIRDsはICUで重篤な意識障害に寄与している可能性があり、診断的な治療トライアル（抗てんかん薬の静注）が検討されることがある。
セクション F　Ictal-Interictal Continuum（IIC） NEW 2021
Ictal-Interictal Continuum

F IICの定義と診断基準

IICとは

IICは「possible ESz（電気的発作の可能性）」または「possible electrographic SE」と同義の純粋なEEG用語であり、診断名ではない。ESzやESEには該当しないが、意識障害・他の臨床症状・神経障害への寄与の可能性が合理的に考えられるパターンである。全臨床的コンテキスト（病歴・身体所見・検査・画像）を踏まえた解釈が必須。

IICの診断基準（すべての条件を満たすこと）

以下のa〜cのうち少なくとも1つを満たし、かつdを満たすこと。

基準内容
a PDsまたはSWが10秒間で平均＞1.0 Hz かつ ≤2.5 Hz（＞10個かつ≤25個/10秒）
b PDsまたはSWが10秒間で平均≥0.5 Hz かつ ≤1.0 Hz（≥5個かつ≤10個/10秒）、かつPlusモディファイア（＋F/R/S）またはFluctuation（変動）を伴う
c 側在性RDA（LRDA, BIRDA, UIRDA, MfRDA等）が平均＞1 Hz かつ ≥10秒間持続（≥10波/10秒）、かつPlusモディファイアまたは変動を伴う（GRDAは対象外）
d 上記のESzまたはESEの基準を満たさない
IICパターンへの抗てんかん薬静注でEEGおよび臨床状態が改善した場合→ECSzまたはECSEの基準Bを満たす。EEGが改善しても臨床改善がない場合→Possible ECSEとなる。
＋α IICの臨床的意義と治療判断

IICは概念的には発展途上であり、広いコンセンサスはない。パターンが意識障害に寄与している可能性がある場合に「少なくともある意味でpotentially ictal（発作の可能性がある）」とみなす考え方。臨床的な治療トライアルを検討する際には、抗てんかん薬の副作用（鎮静・循環抑制）と潜在的ベネフィットを個別に判断する必要がある。

SUMMARY セクション E・F のキーメッセージ

① BIRDsは「short version of seizure」：＞4 Hz・≥0.5秒〜＜10秒のリズミック活動。Evolving / 発作・ED類似形態 / 鋭く輪郭された 3種類のサブタイプ。
② IICは「possible ESz」と同義：ESzには該当しないが発作の可能性が合理的に考えられるパターン。PDs/SW が＞1〜≤2.5 Hz（基準a）、≥0.5〜≤1 Hz＋Plus/変動（基準b）、側在性RDA ＞1 Hz＋Plus/変動（基準c）が対象。
③ IICへの薬剤反応性で分類が変わる：EEG＋臨床の改善→ECSz/ECSE。EEG改善のみ→Possible ECSE。反応なし→IICのまま。
④ GRDAはIIC基準から除外：側在性RDA（LRDA, BIRDA等）がIICに該当しうるが、汎化性RDA（GRDA）は基準cの対象外。
⑤ IICは診断名でなくEEG記述語：全臨床文脈（既往・検査・画像）を踏まえて解釈する。「possible NCSE」と伝える際にも、IICのRPP修飾語での正式記述が推奨される。
セクション G・H　最低限の記録要件とその他の用語
Minimum Reporting Requirements and Other Terms

G 最低限の記録・報告要件

連続EEGモニタリング中の最低限の報告要件

ACNS連続EEGコンセンサス声明（Herman ST et al. 2015[Herman 2015]）の推奨が繰り返し強調されている。

1

開始後30〜60分（標準または日常EEGに相当）：できるだけ早く臨床チームに報告する。

↓
2

24時間ごと：書面での報告を少なくとも1日1回。記録中に著変があれば追加で口頭または書面で報告する。臨床チームへの更新連絡は少なくとも1日2回を推奨（安定している場合を除く）。

H その他の用語：パターン負荷量の定量化

Daily Pattern Burden / Daily Seizure Burden / Daily Pattern Index

発作・パターン負荷量の定量的記述には以下の3指標を使用する。

用語定義例
Daily Pattern Burden 24時間あたりのパターン総持続時間 GPDsが12時間は記録の33%→4時間 ＋ 10時間は10%→1.2時間 ＝ Daily GPD Burden 5.2時間
Daily Seizure Burden 24時間あたりの発作総持続時間 30秒の発作が1日6回→Daily Seizure Burden 3分
Daily Pattern Index Daily Burden × 平均周波数（Hz） = Hz・時間 GPDs 1.5 Hz、Daily Burden 5.2時間→Daily GPD Index 7.8 Hz・時間

時間単位（Hourly Pattern Burden / Hourly Seizure Burden）・最大時間当たり負荷量も同様に算出・報告できる。

SUMMARY セクション G・H のキーメッセージ

① 開始30〜60分で最初の報告：連続EEGモニタリングの開始から30〜60分以内に最初の報告を臨床チームに提供する。
② 24時間ごとに書面報告：著変があれば追加で報告。臨床チームへの更新は少なくとも1日2回（安定している場合を除く）が推奨。
③ Daily Pattern Burden：24時間あたりのパターン総持続時間。発作・パターンの重症度を定量的に表現する重要な指標。
④ Daily Seizure Burden：24時間の発作総持続時間。研究目的の標準化やアウトカム評価に有用。
⑤ Daily Pattern Index = Burden × 周波数：高周波数のパターンはより高いIndexとなり、パターンの「てんかん様強度」を複合的に表す。
