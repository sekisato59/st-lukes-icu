    Q

    細菌感染症で入院中、経過が思わしくないときに考えることは？

    A 吉田先生の回答
    ※ 治療効果判定は原則 48〜72 時間経ってから
    • 治療開始後 1〜2 日しか経過していない場合 → （例外を除いて）経過観察して OK
    • 治療開始から 3 日以上経過して発熱などをきたした場合は、治療失敗の可能性を考える
    ①抗菌薬が効いていない

    培養を出しておく

    ②診断が間違っている
    • 感染源がほかにある 例：細菌性肺炎 → COVID-19 / Flu / 結核 / 肺塞栓 / 器質化肺炎など
    • そもそも感染症ではない他の熱源がある 例：自己免疫疾患 / 血管炎 / 腫瘍熱など
    ③入院中に新規のイベントが起こった
    • CD腸炎
    • 薬剤熱
    • デバイス感染
    • 偽痛風
    ④ドレナージが不十分
    • 膿瘍を形成している 肺化膿症 / 膿胸 / 腎膿瘍 / 肝膿瘍など
    • 喀痰や尿など、排出すべきものが排出できていない
