    感染症各論 ／ Syphilis

    梅毒

    Syphilis（ばいどく）

    RPR / TPHAI〜III 期・潜伏神経梅毒

    病期と臨床像

    梅毒は感染後、I 期（約 3 週）→ II 期（約 3 か月）→ III 期（約 2 年）と進行し、その間に症状のない潜伏期（latent）をとる。

    病期時期主な所見
    I 期 約 3 週
    • 硬性下疳（chancre）：感染部位（多くは陰部）にできる無痛性の硬い潰瘍。痛くないため気づかれにくい
    • 所属リンパ節腫脹：病変近くのリンパ節が腫れる
    II 期 約 3 か月
    • 発熱などの全身症状
    • 肝脾腫
    • バラ疹（梅毒性発疹）：体幹〜四肢、とくに手掌・足底（palms and soles）に出る紅斑。梅毒に特徴的
    • 全身性リンパ節腫脹
    III 期 約 2 年
    • ガム腫（gumma）：皮膚・骨・臓器にできる肉芽腫性のかたまり
    • 大動脈瘤：梅毒性大動脈炎による
    • 脊髄癆（tabes dorsalis）：脊髄後索の障害で感覚失調・歩行障害
    • Argyll Robertson 瞳孔：対光反射は消失するが近見反射は保たれる（光‐近見解離）
    潜伏梅毒 —
    • 早期（early）：感染から 1 年未満
    • 後期（late）：1 年以降、または感染時期不明
    潜伏梅毒（latent syphilis）とは？ 症状はまったくないが、梅毒の血清反応は陽性のままの時期。I 期・II 期の症状が自然に消えた後や、症状に気づかないまま経過した場合がこれにあたる。感染からの期間で 早期（1 年未満）と後期（1 年以降・時期不明）に分け、後期のほうが治療期間が長くなる（→ 治療の項）。

    潰瘍の鑑別

    • 無痛性潰瘍 → 梅毒（硬性下疳）
    • 有痛性潰瘍 → HSV ／ 軟性下疳（chancroid, H. ducreyi）

    血清診断（RPR・TPHA）

    • RPR：non-treponemal test（感染 1〜2 週で陽性化）
    • TPHA：treponemal test（感染 2〜3 週で陽性化）
    RPRTPHA解釈（interpretation）
    ＋＋ 梅毒
    ＋−
    • 初期
    • Biological false positive（妊娠 ／ SLE）
    −＋
    • 治療後
    • TPHA false positive（very rare!）
    −−
    • なし
    • ごく早期
    • Prozone phenomenon（菌量が多すぎて凝集してこない）

    神経梅毒（neurosyphilis）とは

    神経梅毒は、梅毒の病原体が中枢神経系（髄膜・脳・脊髄）に侵入して生じる病態。梅毒のどの病期でも起こりうるのが特徴。とくに ICU・救急では、以下の 3 つを「神経梅毒のサイン」として意識する。

    • ① 眼（ocular：ぶどう膜炎・視力障害など）
    • ② 髄膜炎（meningitis）
    • ③ 耳（otic：難聴・めまいなど）

    神経梅毒の評価フロー（HIV 非感染例）

    神経梅毒を疑ったら、次の 3 ステップで進める。 治療＝神経梅毒として治療する、 なし＝神経梅毒は否定的、という意味。

    1血清トレポネーマ検査でスクリーニング
    • FTA-ABS ／ TPPA ／ TP-EIA ／ CIA のいずれかを確認する（梅毒の感染歴を反映する検査）
    • 陰性 → なし（神経梅毒は考えにくい）
    • 陽性 → ステップ 2 へ
    2症状のパターンで振り分ける
    • 神経症状のみ／無症状（眼・耳症状なし）→ ステップ 3（腰椎穿刺）へ進む
    • 眼症状・耳症状あり（神経症状の有無は問わない）→ まず眼科・耳鼻咽喉科へ紹介
      • 眼梅毒／耳梅毒が確定、または否定できない → 治療（ベースラインの髄液所見も確認しておく）
      • 他疾患で説明でき、神経症状も残らない → なし
      • 他疾患で説明できても神経症状が残る → ステップ 3 へ
    3腰椎穿刺（LP）して髄液で判定
    • 次のいずれかを満たす → 治療
      • CSF-VDRL が陽性
      • CSF 白血球 > 5/μL
      • CSF 蛋白 > 45 mg/dL
    • 3 つすべて満たさない → なし

    NS：neurosyphilis（神経梅毒）。本フローは HIV 非感染例を対象とし、原図の注釈記号（* / ¶ / △）は本文に反映している。
