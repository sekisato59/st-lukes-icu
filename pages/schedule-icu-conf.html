プレゼンテーション

8:30 | ICUカンファレンス

SECTION 1プレゼンテーションの総論

なぜ毎朝のカンファレンスは行われるのか

ICUにおいては、その日にやること（Plan）を明確にし、チーム内で責任を共有するためです。新たに患者のPlanを立てるには、多職種がそれぞれの視点から客観的事実（Objective Data）や状態を評価（Assessment）し、介入点を見つけ出す必要があります。

Planが定まれば、勤務中は優先順位を意識してそれらを遂行していくだけです。一方で、ここが曖昧なままだと、カンファレンス後にあらためて方針を話し合う必要が生じ、結果として時間を浪費してしまいます。日々のカルテ同様にプレゼンでもO → A → Pの流れを意識してカンファレンスに臨むようにしましょう。

STEP 1客観的なデータ（Objective Data）を集められる

「現在、酸素需要が増大し、単純写真で透過性低下を認める。」
「昨日と比べて血小板減少がすすんでいる。」

カルテを見れば誰でも入手できる情報なので、多職種が集まりカンファレンスを開く意義は低そうです。上級医に質問しながらぜひ次のレベルへ！

STEP 2客観的事実を集めて評価（Assessment）できる

「現在、酸素需要が増大し、単純写真で透過性低下を認める。以上から肺炎増悪を疑う。」
「昨日と比べて血小板減少がすすんでいる。感染や消耗性の背景は乏しく新規開始薬剤にPPIがあり、薬剤性を疑う。」

得られた客観的事実から患者の現状を評価できました。一方、これだけでは具体的に何か変更を要するのかが伝わりません。初期研修医のうちに沢山「提案」をできるようになりましょう！

STEP 3客観的データを集めて評価し、プラン（Plan）を提示できる

「現在、酸素需要が増大し、単純写真で透過性低下を認める。肺炎増悪を疑い、培養を採取し抗生剤を●●へ変更する。」
「昨日と比べて血小板減少がすすんでいる。感染や消耗性の背景は乏しく新規開始薬剤にPPIがあり、薬剤性を疑う。オメプラゾールはネキシウムに変更する。」
ICU研修における到達目標です。間違っていても構いませんので、事前に上級医へ内容を確認して、積極的にカンファレンスでプランを提案しましょう！「〜を、検討します」は使わず、プランに具体性をもたせましょう。

SECTION 2By Systemのプレゼンテーション

By System vs By Problem

ICUの朝カンファレンスではBy Systemでの発表を行います。By Problemの評価と比較した長所・短所があるので一度目を通しておきましょう。

By System By Problem
長所 臓器ごとに網羅的に確認でき、漏れが少ない。
プレゼンの流れが定型化し、引き継ぎがスムーズ。		 主要な臨床課題に直結しやすく、意思決定が速い。
診断仮説→介入→反応という臨床推論の流れが明確。
短所 「事実の羅列」「現状報告」になりがち。
プロブレムの優先順位や思考過程が見えにくい。
複数のシステムにまたがることがある。		 ICU患者のようにプロブレムが多いと漏れが生じやすい。
軽微だが重要な臓器異常が後回し／見落とされる可能性。

SECTION 3カンファレンスの流れ

スタッフ医師の司会のもと，以下の順に患者のプレゼンを行います

1.レジデント：By Systemの患者プレゼン（不在時は病棟長が代行）
2.病棟長：レジデントのプレゼンに対して補足
3.インチャージナース：喀痰量、患者の訴え、依頼事項など
4.薬剤師：内服薬再開の有無の確認、薬剤相互作用、血中濃度測定
5.栄養士：栄養計画についてコメント
6.理学療法士：リハビリ計画などについてコメント
7.集中治療科スタッフ：専門医の視点でプレゼンを補足、教育的コメント
  • 患者1人につき5-7分程度で全職種がプレゼンすることを目指します
  • 時間が限られているため冗長なプレゼン内容は削るように心がけましょう

SECTION 4レジデントのICUプレゼンテーションの流れ

Part 1

Opening Statement

患者の全体像を1-2行で + 患者にとっての問題点や大方針 + （経過の長い患者は必要に応じて、直近数日の簡単な経過）

「●●さんは、COPD増悪による呼吸不全で挿管管理を継続している84歳女性です。挿管からすでに2週間が経過しており、明日、気管切開を行った後、一般床退室後に呼吸器のサポート漸減を図る方針としています。」
Part 2

By Systemのプレゼンテーション（3-4分）

各システムについて O → A → P の流れでプレゼンします。各項目をクリックすると詳細へジャンプします。

#1神経系意識 / 鎮静 / 鎮痛 / せん妄↓ #2循環器系昇圧薬 / 体液量 / 組織還流↓ #3呼吸器系挿管 / 非挿管 / HFNC・NIV↓ #4腎電解質系透析 / 血液ガス / 酸塩基↓ #5消化器/肝胆道系腹部所見 / 肝機能 / ドレーン↓ #6血液系貧血 / 凝固 / 血小板↓ #7感染症抗菌薬 / 培養 / フォーカス↓ #8栄養/代謝系経腸栄養 / 血糖管理↓ #9筋骨格系リハビリ / ICU-AW↓ #10予防DVT / 潰瘍 / VAP / 褥瘡↓
  • 該当しないシステムについては「介入なし」として構いません
  • 問題点が複数のシステムにまたがる場合には、1箇所にまとめてよいです
  • Social Problemsは発表不要。ただし患者家族とのトラブルや社会調整を要する場合は発表に含めて共有を
Part 3

Closing：To Doの確認

By Systemのプレゼンで挙げたPlanを再度まとめる。最後でもう一度、今日やるべきことをチームにリマインドする。

SECTION 5By Systemのプレゼンテーション

1
Neurological System
神経系
N
意識レベル
  • 非挿管患者：GCS（Glasgow Coma Scale）
  • 挿管患者：RASS（Richmond Agitation-Sedation Scale）
FREQUENT DISCUSSION POINT

無鎮静で意識障害があるなら、それはなぜ？？
鎮静薬を使用していないにもかかわらず意識レベルが低下している場合、その原因を必ず考察しましょう。代謝性脳症、感染症、頭蓋内病変など、見逃してはならない病態が隠れている可能性があります。

N
鎮静
  • 使用する薬剤の種類、投与量は？
    • - ミダゾラム：mg/h または mL/h（組成：1mL=1mg）
    • - プロポフォール：mg/h または mL/h（組成：1mL=10mg）
    • - デクスメデトミジン：mL/kg/h（組成：1mL=100μg）
    • - モルヒネ：mg/h（組成：症例に依る）
  • 本日、目標とする鎮静深度は？ → 浅鎮静、深鎮静ではなくRASSで表現しよう
FREQUENT DISCUSSION POINT

そもそも患者に鎮静は必要か？

N
鎮痛
  • 疼痛の原因は？
  • 疼痛の評価は？
    • - 非挿管患者：NRS（Numeric Rating Scale）
    • - 挿管患者：BPS（Behavior Pain Scale）
  • 使用する薬剤の種類、投与量は？
    • - レミフェンタニル：γ（組成：1mL=100μg/mL）
    • - フェンタニル [持続点滴]：μg/h（組成：1mL=50μg）
    • - フェンタニル [IV-PCA]：μg/h（組成：1mL=10μg）
N
せん妄・不眠
  • せん妄はあるか？ → ICU患者ではCAM-ICUで評価
  • せん妄の原因は？誘発因子は？
  • 使用する抗せん妄薬の種類、投与量は？
  • 不眠の原因は？使用する眠剤の種類、投与量は？
N
脳外科患者
  • てんかん、痙攣：抗痙攣薬の種類・投与量・増量可能な時期、抗浮腫薬、脳波所見
  • 頭部外傷：IPC上限、CPP目標値、二次性脳損傷予防
二次性脳損傷予防チェックリスト
  • 頭部を15-30°挙上、頸部正中位
  • 酸素療法 PaO2 > 60mmHg
  • 高体温の予防
  • 血糖値 110-180mg/dL
  • CO2管理目標値：PCO2 30-40mmHg
2
Cardiological System
循環器系
C
昇圧薬・降圧薬
  • 血圧の管理目標は？
    • - 外科術後（特に脳外科）は収縮期血圧を低めに管理
    • - 逆に高血圧の患者は、昇圧剤使用時に目標血圧を高く設定することも
  • 昇圧剤需要の原因は？ → 真のショック or 鎮静薬の副作用によるカウンター
  • 昇圧薬/降圧薬の種類、投与量は？
    • - ノルアドレナリン：γ / ピトレシリン：U/h / ニカルジピン：γ / ニトログリセリン：γ
  • 組織還流は保たれているか？ → CRT、乳酸値の推移、皮膚所見、意識、尿量
C
体液量

体重、浮腫、心エコー、画像所見、CVPなどを総合的に評価する。

  • 体液量過多：体液量を減らす状況か？昇圧剤使用下での有効循環血漿量を減らす介入は慎重な判断を
  • 体液量正常：ドライサイドに管理する必要はあるか？（抜管前など）
  • 体液量減少：輸液チャレンジ / PLR / VTI / LiDCOで評価。輸液耐性はあるか？
3
Respiratory System
呼吸器系
R
挿管患者
  • 呼吸器設定は適切か？ → モード（PCV, VCV, PSV）、肺保護換気を達成できているか？
  • 患者の状態は？ → P/F, pCO2のトレンド、非同調の有無、適切な鎮静・鎮痛
  • 人工呼吸器管理を継続するか？
    • - 継続する：抜管の律速は？挿管してから何日目か？気管切開の予定は？
    • - 抜管する：SAT/SBTクリア？カフリークテスト？ステロイド？抜管後のデバイス？経腸栄養の中止？
R
非挿管患者 / HFNC・NIV
  • 呼吸様式、呼吸音は問題ないか？ → 血液ガス所見が問題なくとも、その呼吸様式を保てるか？
  • HFNC/NIVのモード・設定・装着の目的は？
  • 治療効果の評価：HACOR score（> 5で失敗予測）、ROX index（< 4.88で失敗予測）
  • デバイスダウンは可能か？ → 離脱プロトコールはICUデジタルの資料を参照
4
Nephrological System
腎電解質系

腎電解質系の評価

  • 透析患者 → 詳細はICUコアカンの「透析患者のプレゼンテーション」を参照
  • 血液ガス所見
    • - Boston法でもよいが、ICUではStewart法で評価すると患者の病態を理解しやすいことがある
  • 代謝性アシドーシス？
    • - AG開大性代謝性アシドーシス → 補正HCO3を求め、隠れた代謝性変化を見つける
  • 低ナトリウム血症 → 低Na血症の原因はなにか？
