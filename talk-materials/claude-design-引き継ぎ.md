# Claude Design 引き継ぎセット

このファイルは、`kanto-ai-education-2026.md`（スライド設計図）を **Claude Design でスライド化**するための指示書です。

## 使い方（3ステップ）
1. 下の「① Claude Designへの指示文」をコピーして Claude Design に貼る。
2. 続けて `kanto-ai-education-2026.md` の **「# 本編スライド（全21枚）」〜「## S備」までの本文**を貼る（設計ログ・付録は貼らない）。
3. 生成後、〔要・実データ〕〔要確認〕の箇所に実素材（NotebookLM成果物・Before/After数値・Canva機能名など）を差し替える。

## ① Claude Designへの指示文（コピペ用）

```
次のMarkdownを、医学会の教育講演スライド（20〜30分・21枚＋備え1枚）に変換してください。

【全体トーン】
- 誠実・控えめ・専門家ぶらない。クリーンで余白を広く。1スライド＝1メッセージ。
- 聖路加国際病院ICUのブランドに合う、落ち着いた配色。ベースは深い紺〜ティール、背景は白〜オフホワイト、アクセントは1色に絞る。多色・派手な装飾・影は使わない。
- フォントは Noto Sans JP。見出しは太字で大きく、本文は短い箇条書き。文字を詰め込みすぎない（1枚3〜5項目以内）。
- 絵文字は多用しない（アイコン化は控えめに可）。テキストボックス左端の色帯は使わない。

【変換ルール】
- 各「## Sn ｜ …」を1枚のスライドにする。見出し＝スライドタイトル、直後の**太字行**＝キーメッセージ（大きめ）。
- 「> 話者ノート」は presenter notes に入れ、スライド面には表示しない。
- タイトル末尾の〔初級／中級／上級／全員〕は、隅の小さなチップ（ラベル）にする。
- 表（| … |）はそのまま見やすい表組みに。
- 〔要・実データ〕〔要確認〕〔※…〕はプレースホルダとして残す（後で差し替えるため）。

【特に印象的にしたいスライド】
- S6（スローガン「好きなものを、学びたいときに、学びたいだけ」）と S21（締め）は、文字大きめのフルブリードで余韻を持たせる。
- S5（作る側の負担）は「テーマ→文献→スライド→進行→振り返り」を横一列の矢印フローにする。
- S11（作るAI／見張るAI）は左右2カラムの対比図にする。
- S18（Before/After）は2列の対比表で「以前→今」を強調。

【言葉づかい】
専門用語は避ける。スライド面に「ハルシネーション」「LLM」「監視エージェント」等は出さない（本文では既にやさしい日本語に言い換え済み）。
```

## ② デザインの早見表（Claude Designが迷ったとき用）

| 要素 | 指定 |
|---|---|
| 配色ベース | 深い紺〜ティール（聖路加ICU）／背景は白〜オフホワイト |
| アクセント | 1色のみ（例：ティールまたは差し色の橙を要所だけ） |
| フォント | Noto Sans JP（見出し太字／本文レギュラー） |
| 1枚の情報量 | タイトル＋キーメッセージ＋箇条書き3〜5 |
| レベルチップ | 〔初級〕〔中級〕〔上級〕〔全員〕を隅に小さく |
| 禁止 | 多色・影・絵文字多用・左端の色帯・詰め込み |
| 話者ノート | presenter notes（スライドに出さない） |
