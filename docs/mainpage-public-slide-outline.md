# 聖路加ICUメインページ 一般公開紹介スライド 骨格案

## 全体方針

- 想定対象: ICUローテーター、研修医、レジデント、院内スタッフ、公開ページを初めて見る人
- ゴール: 「どこに何があり、困ったときにどう探すか」が3分でわかる
- トーン: Webサイトの機能紹介というより、ICUでの学習・業務を支えるポータルの紹介
- ビジュアル方針: 各章の最初に実画面スクショを大きく配置し、操作説明は短い吹き出しで重ねる
- 現在のコンテンツ規模: 全ページ一覧 295件、新着一覧 46件、検索インデックス 1351項目

### 見出し

### スクショ

## 4. 検索機能の使い方

### 見出し
探したい言葉を入れるだけ

### 操作ステップ
1. トップページ右上または新着欄の「コンテンツをすべて検索」をクリック
2. キーワードを入力
3. 必要に応じて「ジャンル」「種別」「System」で絞り込み
4. タイトルをクリックして該当ページへ移動

### 検索例
- 「抗菌薬」
- 「ARDS」
- 「B型肝炎」
- 「透析」
- 「RASS」

### スクショ
- pages/recent-all.html の検索バー
- キーワード入力後の検索結果
- ジャンル、種別、Systemのプルダウンを拡大

## 9. よくある場面別の使い方

### 見出し
困った場面から逆引きする

### シナリオ例
- ローテ初日: 「ローテされる先生へ」→ スケジュール → ICUのルール
- 患者入室時: FAQ → ICU入室時To Do → 関連ページへ
- 抗菌薬の用量に迷う: 便利ツール → 抗菌薬投与量 一発計算
- 論文・GLを探す: 全ページ検索 → ジャンル「論文・GL」→ System絞り込み
- 根拠を確認したい: 全ページ検索 → 論文・GL → System絞り込み

### スクショ
- 1スライド内に小さめの画面キャプチャを4枚並べる
- それぞれに「入口」「入力」「結果」「詳しく読む」のラベル
